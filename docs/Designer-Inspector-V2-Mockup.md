## 1. 整体布局

- **右侧字段区**：每个分类显示对应字段，两列布局（标签 | 编辑控件）

## 3.「动画」Tab — 6 类标准动画

### 3.1 左导航

```
┌─动画─────────┐
│ 总览          │   ← 列出所有已配置动画
│ ──变量连接── │
│   读取地址    │   ← 等同于现"数据绑定"
│ ──显示── 　 │
│ + 外观        │   ← 加号 = 添加
│ + 可见性      │
│ ──移动── 　 │
│ + 水平移动    │
│ + 垂直移动    │
│ + 直接移动    │
│ + 对角线移动  │
└──────────────┘
```

### 3.2 外观动画（范围式）

```
┌─ 外观 ──────────────────────────────────────────────┐
│  变量                                                │
│    名称  [DB1050_DriveControl.AxisCtrl[1].Cmd.Teac…]│
│    地址  ns=4;s=|var|...   (只读)                   │
│                                                      │
│  类型   ● 范围   ○ 多个位   ○ 单个位                │
│                                                      │
│ ┌────┬────────────┬────────────┬────┐   ↑↓ × ＋    │
│ │范围│ 背景色      │ 前景色      │闪烁│              │
│ ├────┼────────────┼────────────┼────┤              │
│ │ 0  │ █ 灰  ▼    │ █ 黑  ▼    │ 否 │              │
│ │ 1  │ █ 绿  ▼    │ █ 黑  ▼    │ 否 │              │
│ │ 2  │ █ 黄  ▼    │ █ 黑  ▼    │ 是 │              │
│ │ ≥3 │ █ 红  ▼    │ █ 白  ▼    │ 是 │              │
│ └────┴────────────┴────────────┴────┘              │
└──────────────────────────────────────────────────────┘
```

**逻辑**：每行代表一个**触发条件**——
- "范围"模式：变量值落在哪个区间，用该行的颜色和闪烁
- "多个位"模式：值的某些位为 1 时匹配
- "单个位"模式：第 N 位为 1 时匹配

支持上下移序、删除、新增。

### 3.3 可见性动画

```
┌─ 可见性 ────────────────────────────────────────────┐
│  变量    [DB1000.Cmd.Show         🔶]              │
│  条件   ○ 变量为 True 时显示                        │
│         ● 变量为 False 时显示                       │
│         ○ 值在范围 [_____] - [_____] 时显示         │
│  否则   ● 隐藏  ○ 禁用（灰色）                     │
└──────────────────────────────────────────────────────┘
```

### 3.4 水平移动 / 垂直移动 / 直接移动 / 对角线移动

```
┌─ 水平移动 ──────────────────────────────────────────┐
│  变量    [DB1000.AxisPos.X      🔶]                │
│  变量范围  从 [0     ] 到 [1000  ]                  │
│  位置范围  起 X [9   ⇅]  终 X [600 ⇅]               │
│  Y 不变（保持原 Y）                                 │
└──────────────────────────────────────────────────────┘
```

**直接移动**：X、Y 各绑一个变量，像素 = 变量值（适合显示坐标）。
**对角线移动**：同时配 X 和 Y 的两段映射。

## 6. 数据模型升级

### 6.1 当前（V1）

```csharp
WidgetInstance {
    string ActionType;     // "set-on" / "navigate" / ...
    string ActionParam;    // "DB1000.Cmd|True"
    Binding Binding;       // 读地址
    List<Animation> Animations;  // 当前简单规则
}
```

### 6.2 新模型（V2）

```csharp
WidgetInstance {
    Binding ReadBinding;  // 主读地址（保留旧字段名映射）
    
    Dictionary<string, List<ActionStep>> Events;
    // key:   "click" / "press" / "release" / "activate" / "deactivate" / "valueChanged"
    // value: 该事件下顺序执行的动作列表

    AppearanceAnimation? Appearance;   // 范围式：(从值, 到值, BG, FG, blink)[]
    VisibilityAnimation? Visibility;   // 条件→显示/隐藏
    MoveAnimation? Movement;           // 单选一种 movement
}

class ActionStep {
    string FunctionId;          // "set-bit" / "reset-bit" / "navigate" / "write-int" / ...
    Dictionary<string,string> Args;  // 命名参数表
}

class AppearanceAnimation {
    string TagId;
    AnimationMatchType Type;  // Range / MultiBit / SingleBit
    List<AppearanceRow> Rows;
}
class AppearanceRow {
    string RangeFrom, RangeTo;  // Range 模式
    int BitIndex;               // SingleBit 模式
    string Background, Foreground;
    bool Blink;
}
```

### 6.3 向后兼容

序列化加载时：
- 旧 `ActionType+ActionParam` → 自动迁移到 `Events["click"] = [{FunctionId=ActionType, Args=parsed}]`
- 旧 `Animations` 列表 → 拆分到 `Appearance` / `Visibility`（按 `TargetProperty`）

存盘时使用新格式，旧文件读得进、新文件旧版本读不进（可加 schema version 标识）。

## 7. 关键交互逻辑

### 7.1 字段动态化（绑定到变量）

每个属性字段右侧的 🔶 图标：
- 灰色 = 静态（默认）
- 橙色 = 已绑变量

点击 → 弹出小卡片：
```
┌─动态化──────────────────┐
│ ● 静态值                │
│ ○ 绑定变量               │
│   变量 [_______ 🔍]      │
│ ○ 绑定脚本               │
│   函数 [_______ ▼]       │
│ [取消]    [确定]         │
└─────────────────────────┘
```

### 7.2 从变量树拖到画布

- 拖一个 BOOL → 弹「创建为：按钮 / 指示灯 / 开关」
- 拖一个 INT/REAL → 弹「创建为：I/O 域 / 棒图 / 量规 / 数值显示」
- 拖一个 STRING → 弹「创建为：文本域 / 符号 I/O 域」
- 拖一个 ARRAY → 弹「创建为：表格 / 趋势 / 数组指示器」

控件自动定位到鼠标释放点、自动完成读地址绑定。

### 7.3 多选编辑

选中多个 widget 时：
- Inspector 只显示**交集属性**（如位置、颜色、字体）
- 修改一处，所有选中对象同步
- 不一致的字段显示为空 + 灰色占位 "—— 多个值 ——"
