## 已知限制

- 状态动画规则修改后需要重选控件以重新注册 Tag（实现按需重建）
- 状态动画当前不支持"恢复默认值"（条件为 false 时）—— 需要用户加反向规则
- 全屏模式不阻止 Alt+Tab / Win 键（仅隐藏 UI）
- Tag 浏览器是模态对话框；未来可改为侧边面板
- 模板页本身在设计器中正常显示，但运行时不出现在页签 —— 用户可能感觉不一致（注意提示）
- 页面模板叠加是单层（不支持嵌套模板）
- WidgetAnimation 不支持多条件组合（AND/OR），单条规则一对一
